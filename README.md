# Blog Site

## Requirements
- NodeJS
- MySQL, MySQL Workbench running locally
    - Alternatively run app with an environmental variable of **'BLOG\_DB\_HOST'**
    - MySQL schema with the name of *'blogsite'*
    - User defaults to *'root'*
    - Password defaults to *'password'*
- Patience

## How to Run the app
1. `git pull <this-url>`
2. `npm i`
3. `npm start`
